## O que é uma ALU?

A **Unidade Lógica e Aritmética** (ALU — *Arithmetic Logic Unit*) é o componente central de qualquer processador. É ela quem realiza todas as operações de cálculo: somas, subtrações, comparações, operações bit a bit e deslocamentos de bits.

Em um processador, o ciclo de execução de uma instrução segue o caminho:

```
Busca → Decodificação → Execução (ALU) → Escrita de resultado
```

A ALU recebe dois operandos (A e B), executa a operação selecionada pelo opcode e devolve o resultado — junto com flags como carry-out, overflow e sinal negativo.

```
        ┌─────────────────────┐
A[7:0] ─►│                     ├─► Resultado[7:0]
        │       ALU 8-bit      │
B[7:0] ─►│                     ├─► Flags (Cout, Neg, Over)
        │                     │
SEL[2:0]─►│  Seletor de Operação│
        └─────────────────────┘
```

## Estrutura desta ALU

Esta ALU foi implementada do zero no simulador **Digital** usando arquitetura hierárquica: cada operação é um módulo `.dig` independente, e o módulo principal `ALU8bits.dig` usa um **multiplexor 8:1** controlado por 3 bits de seleção para rotear a saída correta.

## Hierarquia de módulos

```
ALU8bits.dig
├── soma8bits.dig
│   └── soma1bit.dig          (somador completo de 1 bit)
├── sub8bits.dig
│   ├── inversor8bits.dig
│   └── soma8bits.dig
├── subWRestore8bits.dig
│   └── sub8bits.dig
├── multiplicacao8bits.dig
│   └── soma1bit.dig
├── divisor8bits.dig
│   └── subRestore1bit.dig
│       └── soma1bit.dig
├── LeftShift8bits.dig
├── RightShift8bits.dig
├── nand8bits.dig
└── XOR8bits.dig
```

Todos os módulos trabalham com **barramento de 8 bits** e usam **splitters** para decompor os operandos em bits individuais quando necessário — especialmente nas operações de multiplicação e divisão, onde o processamento é feito bit a bit.

:::info Sobre o simulador Digital
Os circuitos foram construídos no [Digital](https://github.com/hneemann/Digital), um simulador de circuitos lógicos open-source. Os arquivos `.dig` são XML que descrevem a topologia do circuito: componentes, posições e conexões.
:::
